## Cheatsheet for Intellij Idea Shortcuts

### Basic Keybindings

```
Open Project Tab - cmd + 1

Run project - ctrl + r

Show code completions - ctrl + space

Show code improvement suggestions - opt + enter

Search everywhere - shift twice

Search for actions - shift + cmd + a

Search for classes - cmd + o

Search of symbol - cmd + opt + o

Selection - opt + up arrow

Duplicate line below - cmd + d

move line - opt + shift + up/down arrow

move functions - cmd + shift + up/down arrow

collapse function - cmd + -

expand function - cmd + =

collapse/expand all functions - cmd + -/=

surround the selected code with something - cmd + opt + t
```
